Validação

O que é Validação?

A validação é quando você apresenta seu site para a pessoa que o solicitou, recebendo sua avaliação, alinhando expectativas e bojetivos.

Como Validar?

  • 1- Tenha o projeto "pronto"
  • 2- Apresente-o para o dono
    do projeto
  • 3- Alinhe metas e objetivos
  • 4- Repita!

Por que validar?

Validar o desenvolvimento de um projeto é fundamental, pois dessa forma, o dono do projeto indica as mudanças a serem feitas.

Dessa forma o produto final fica de acordo com as expectativas, todos ganham!.
