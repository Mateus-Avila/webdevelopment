Desenvolvimento

O que é Desenvolvimento?

A fase de desenvolvimento é onde a equipe de programadores traduz o design e os protótipos em código real, utilizando tecnologias e linguagens de programação.

Como Desenvolver?

  • 1- Tenha o protótipo em mãos
  • 2- Defina as tecnologias a
    serem utilizadas
  • 3- Programe!

Tecnologias utilizadas:

HTML: High Text Markup Language - Linguagem de marcação de texto (Estrutura)

CSS: Cascade Styling Sheets - Folhas de estilo em cascata (Personalização)

JavaScript: Linguagem de programação que traz iteratividade para a página.

Cursos Sugeridos:

  • 1- HTML5 e CSS3 - Curso em Vídeo
  • 2- Formação HTML e CSS Alura
  • 3- Formação JavaScript Alura
