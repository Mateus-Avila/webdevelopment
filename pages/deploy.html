Deploy

O que é Deploy?

Deploy é o processo de disponibilizar o site ao público. Isso envolve a transferência do código-fonte e dos recursos necessários para que a aplicação funcione corretamente em um servidor ou plataforma acessível aos usuários finais.

Como fazer o Deploy?

  • 1- Tenha o projeto "pronto"
    e em um repositório
  • 2- Utilize sua plataforma de
    hospedagem favorita
  • 3- Faça o Deploy do Site
  • 4- Mostre para os outros!

Vercel é um site onde você consegue fazer o deploy do seu site de forma gratuita! Sendo de muito fácil utilização.

Com ele, você pode mostrar o seu site para outras pessoas!.
